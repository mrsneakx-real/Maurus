# Install and Run instructions

Step 1: Download the current Maurus version from this repository.<br>
Step 2: Run the installer **as an administrator.**<br>
Step 3: After the installation create a desktop shortcut to Maurus.exe. **IMPORTANT: The shortcut has to run Maurus.exe as an administartor.**<br>
Step 4: Lastly run the shortcut. Now Maurus runs on your Pc.<br>
